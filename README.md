# HA (High-Availability) InfluxDB

## Start InfluxDB 1

```shell
make influx1_up
```

## Start InfluxDB 2

```shell
make influx2_up
```

## Start Relay 1

```shell
make relay1_up
```

## Start Relay 2

```shell
make relay2_up
```

## Start Load Balancer

```shell
make loadbalancer_up
```
